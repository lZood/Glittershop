# **App Name**: Glittershop

## Core Features:

- Product Catalog: Display a catalog of luxury jewelry products, fetching data from Supabase DB (PostgreSQL).
- User Authentication: Enable user registration, login (with email/password and providers like Google/Apple), and session management using Supabase Auth.
- Secure Payment Processing: Integrate Stripe/Mercado Pago via Supabase Edge Functions to securely process payments.
- Intelligent Search: Sync the PostgreSQL database with Algolia for powerful product search functionality.
- Optimized Image Hosting: Host and serve product images in an optimized manner using Supabase Storage.
- Transactional Emails: Send transactional emails (e.g., order confirmations) using SendGrid, invoked from Edge Functions.
- Interactive Gift Guide: Generate a tool to guide users with suggestions for items that may interest them.

## Style Guidelines:

- Primary color: Deep blue (#3B5998) to evoke a sense of luxury and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern backdrop.
- Accent color: Gold (#FFD700) to highlight key elements and call attention to luxury.
- Headline font: 'Playfair' (serif) for an elegant, fashionable feel.
- Body font: 'PT Sans' (sans-serif) for readability and a modern touch; pairs well with 'Playfair'.
- Use minimalist, line-based icons to maintain a sophisticated aesthetic.
- Prioritize a mobile-first, clean layout with ample white space to showcase the jewelry.
- Incorporate subtle, elegant transitions and animations for product display and user interactions.
